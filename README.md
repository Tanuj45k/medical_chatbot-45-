# Medical Chatbot

This is a rule-based medical chatbot built using Python. It allows users to input symptoms and get basic guidance on possible conditions.

## Technologies Used
- Python
- Flask
- HTML/CSS
- Bootstrap

## Features
- Symptom-based interaction
- Voice input support
- Responsive UI
